# FEDC4_TMI_HOMERS_OFF

## 프로젝트 세팅 방법

다음 명령어를 입력하면 프로젝트 설치 후 실행됩니다.

```bash
git clone https://github.com/prgrms-fe-devcourse/FEDC4_TMI_HOMERS_OFF.git && cd FEDC4_TMI_HOMERS_OFF
npm install
npm run dev
```
